ARCHITECTURE

The Technology Stack

Feather MD is built cleanly with modular layers that respect both the host operating system resources and web security boundaries.

Shell & Core

Tauri 2 (Rust)

Leverages native system webviews to deliver lightweight binaries, lightning-fast execution, and near-zero idle CPU cycles.

Editing Area

CodeMirror 6

A modular, tree-shaken editor core supporting syntax highlighting, brackets auto-pairing, code folding, and optional Vim keys.

Compilation Pipeline

Marked + DOMPurify

Compiles Markdown synchronously into native HTML, coupled with DOMPurify for absolute, client-side XSS protection.

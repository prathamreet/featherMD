About Feather MD

Feather MD v1.6.1

Pure Performance Markdown Editor

Built with Tauri + Rust + Vanilla JS
Memory: < 60MB RAM
Cold Start: < 100ms

Engineered to write. Built to fly.

01

Bidirectional Scroll Sync

Scroll either the CodeMirror 6 editor or the preview pane, and the other pane follows perfectly with ratio-based tracking.

02

Advanced Printing Engine

Bypasses browser native limits to print beautiful, clean, multi-page documents without hostnames or page URLs.

03

Signed Auto-Updates

Release packages are cryptographically signed with Ed25519 for absolute security, updating seamlessly on launch.

04

OS File Association

Integrates directly with Windows and Linux. Double-click any markdown file to launch instantly from your file manager.

05

Premium Unified Header

Combines window titlebar and editor toolbar into a singular, mathematically centered bar with floating interactive tooltips.

06

10 Built-In UI Themes

Five light and five dark themes. Switch instantly under 1ms with custom repainting locked to CSS root variables.

07

External File Watcher

Monitors active files in the background using tokio async channels. Prompting you instantly on external alterations.

08

Keyboard-First Workflows

Full command panel shortcuts for fast toggles of line numbering, word wraps, active-line highlights, and find/replace.

The Technology Stack

01
Shell & Core

Tauri 2 (Rust)

Natively compiled Rust core utilizing host webviews to ship a secure, 5 MB package with zero Electron runtime bloat.

02
Editing Area

CodeMirror 6

Modular, tree-shaken editor core supporting active-line highlighting, quote auto-pairing, folding, and custom Vim keys.

03
Compilation Pipeline

Marked + DOMPurify

Synchronous Markdown compiler coupled with DOMPurify XSS protection and Highlight.js on-demand syntax loading.
